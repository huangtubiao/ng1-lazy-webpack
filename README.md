# ng1-lazy-webpack

在此分享一个基于 `angularjs + webpack + ocLazyload` 的 **Angularjs 异步模块加载项目模板**。
---
## Dependences

- node^6.2.0
- npm^3.8.9
- angular^1.4.8

## Getting Started

- npm install
- npm run dev

## 项目文件夹结构

- app: 用于存放开发代码
- build: 用于存放打包后代码
- dist: 用于存放生产环境代码
